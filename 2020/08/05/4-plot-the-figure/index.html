Plot the figure

Preface

R packages, codes, and my aesthetic.

Plot chromosomes/genes

Some softwares:

(in random order, recommended ones in bold font)

  • RIdeogram

  • gggenes

    • R package (gg style)
    • for local genes
    • looks pretty

  • ggbio

    • R package (gg style)

    • for local genes

    • simple, but not bad

    • version hg19

    • really takes time to install

    • lots of packages are required, (my way to use ggbio)

      library(ggbio);

      library(gridExtra);

      library(gridBase);

      library(reshape2);

      library(Homo.sapiens);

      library(EnsDb.Hsapiens.v75);

      library(BSgenome.Hsapiens.UCSC.hg19);

      library(TxDb.Hsapiens.UCSC.hg19.knownGene);

      data(genesymbol, package = “biovizBase”);

      library(ggpubr)

  • Gviz

  • PhenoGram

    • Ruby language
    • whole genome
    • GWAS catalog style
    • don’t support PDF (pity)
    • should manually modify the “pheno_gram.rb” script to change colors
      • line ~650, “class ExhaustiveSearchColorMaker”
    • also provide web server

  • ChromPlot

    • R package
    • whole genome
    • a lot of functions
    • don’t know how to change the font size (too small)
    • looks ugly

  • Tagore

    • Python module (python 3+)
    • whole genome
    • easy to use
