Tutorial to deploy hexo to github page

in an easy way

Install node

Download nodejs from https://nodejs.org/zh-cn/download/
Install

Install Hexo

$ sudo npm install -g cnpm --registry=https://registry.npm.taobao.org
$ cnpm install hexo-cli -g

Initiation

$ hexo init panyuwen.github.io ## same username as guthub

Install

$ cd panyuwen.github.io
$ cnpm install

Run

$ hexo clean ## clean cache
$ hexo g  ## generate static page
$ hexo s  ## run on server

Open in browser

Direct to localhost:4000

Modify configuration file

Open _config.yml

Change the Hexo theme

$ git clone https://github.com/theme-next/hexo-theme-next themes/next

Then modify _config.yml
    theme: next

Re-run

$ hexo g
$ hexo s

Write an article

$ hexo new ArticleName

Then open source/_posts/ArticleName.md and write

Tags for article

$ hexo new page tags  ## to start the function

Modify _config.yml
    tag_dir: tags
Modify source/tags/index.md
    ---
    title: tags
    date: xxx
    type: "tags"
    comments: false
    ---

Categories for article

$ hexo new page categories

Modify _config.yml
    category_dir: categories
Modify source/categories/index.md
    ---
    title: categories
    date: XXX
    type: "categories"
    comments: false
    ---
Put categories & tags in the article
    ---
    title: xxx
    date: xxx
    categories:
    - category
    tags:
    - tag1
    - tag2
    ---

Deploy to Github

Start a new public repository, named panyuwen.github.io

Install hexo-deployer-git
$ cnpm install hexo-deployer-git --save

Modify _config.yml
deploy:
  type: git
  repo: https://panyuwen:{yourpassword}@github.com/panyuwen/panyuwen.github.io.git
  branch: master

$ hexo clean; hexo g; hexo d  ## deploy, might be unstable in China. if fails, try again

Open in browser

Direct to https://panyuwen.github.io/

Create, modify, and delete articles

$ hexo new ArticleName
$ vim source/_posts/ArticleName.md
$ rm source/_posts/ArticleName.md
$ hexo clean; hexo g; hexo d
