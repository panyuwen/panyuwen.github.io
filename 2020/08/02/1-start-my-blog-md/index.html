Welcome!

This is my first post, I’m going to share the way how I build my web.

Before we start, you should have your own github account, since all the codes were deployed to github. Besides, you should have “git” installed on your computer, and it’s better to know some basic commands of git. You can find anything your want to know on the internet. I think my second post would also be helpful.

Tutorial to deploy hexo to github page

To make it work

Install Nodejs & Hexo

## download and install nodejs on your computer, https://nodejs.org/zh-cn/download/
## then install hexo on your computer
$ sudo npm install -g cnpm --registry=https://registry.npm.taobao.org  
## use cnpm from taobao could speed up the following steps
$ cnpm install hexo-cli -g

Initiation, install, & run

$ hexo init panyuwen.github.io ## same username as your guthub account
$ cd panyuwen.github.io
$ cnpm install
$ hexo clean ## clean cache
$ hexo g     ## generate static page
$ hexo s     ## run on server

## then open your browser
## direct to localhost:4000
## if you see a web page, it's working
## then let's make it a informative and useful web

Modify configuration file

$ vim _config.yml
## change title, and so on

Change the Hexo theme

$ git clone https://github.com/theme-next/hexo-theme-next themes/next

## then modify _config.yml, change the "theme"
##   theme: next

$ hexo g; hexo s  ## re-run to enjoy your new web

Write an article

$ hexo new ArticleName
$ vim source/_posts/ArticleName.md

Tags & Categories for your article

$ hexo new page tags  ## to start the function
$ hexo new page categories

## modify _config.yml
    tag_dir: tags

## modify source/tags/index.md
    ---
    title: tags
    date: xxx
    type: "tags"
    comments: false
    ---

## modify _config.yml
    category_dir: categories

## modify source/categories/index.md
    ---
    title: categories
    date: XXX
    type: "categories"
    comments: false
    ---

## add categories & tags to the article
    ---
    title: xxx
    date: xxx
    categories:
    - category
    tags:
    - tag1
    - tag2
    ---

Deploy to Github

## now, it's time to show your web to the whole world
## start a new public repository on github, named panyuwen.github.io
## install hexo-deployer-git (the following command)
$ cnpm install hexo-deployer-git --save

## modify _config.yml, to add your github account, so that your machine knows where to deploy your codes
## there're 2 ways to avoid typing your password again and again

## 1. 
## copy your SSH key in ~/.ssh/id_rsa.pub
## in your github account, click your mouse, your avatar --> setting --> SSH and GPG keys --> new SSH key --> paste & save
## if you don't have your SSH key, use the command "ssh-keygen -t rsa" to generate
## if you think its difficult, you can just turn to way 2
## if you have succeed in adding your SSH key to your github account, then modify _config.yml
    deploy:
      type: git
      repo: git@github.com:panyuwen/panyuwen.github.io.git
      branch: master

## 2.
    deploy:
      type: git
      repo: https://panyuwen:{your_password}@github.com/panyuwen/panyuwen.github.io.git
      branch: master

## deploy
## might be unstable in China
## if it fails, just try again
$ hexo clean; hexo g; hexo d  

## then open your browser
## direct to https://panyuwen.github.io/

Create, modify, and delete articles

$ hexo new ArticleName  ## create
$ vim source/_posts/ArticleName.md  ## modify
$ rm source/_posts/ArticleName.md  ## delete
$ hexo clean; hexo g; hexo d  ## deploy

To make it fancy

We are going to modify themes/next/_config.yml (default)

Dynamic Background

$ git clone https://github.com/theme-next/theme-next-canvas-nest themes/next/source/lib/canvas-nest

canvas_nest:
  enable: true
  count: 96

Sidebar Settings

sidebar:
  position: right
  display: always

avatar:
  url: /images/IMG_F7DE10EB3810-1.jpeg  ## place the photo under theme/next/source/images/
  rounded: true  ## the avatar will be dispalyed in circle

social:
  GitHub: https://github.com/panyuwen || fab fa-github
  E-Mail: mailto:panyuwen.x@gmail.com || fa fa-envelope

Post Settings

reward_settings:
  enable: true
  animation: false
  comment: Peace & Love

reward:
  wechatpay: /images/wechat.JPG
  alipay: /images/ali.JPG
  ## put your QR code under theme/next/source/images/

Search Services

$ cnpm install hexo-generator-searchdb --save

local_search:
  enable: true

## open _config.yml, and add
search:
  path: search.xml
  field: post
  format: html
  limit: 10000

Site Information Settings (copyright)

powered: false
theme: flase

creative_commons:
  license: by-nc-sa
  sidebar: true
  post: true

## then open _config.yml
# URL
url: https://panyuwen.github.io/

Math Formulas Render Support

$ cnpm install hexo-math --save

## modify themes/next/_config.yml
math:
  enable: true
  per_page: true
  engine: mathjax
mathjax:
  enable: true
  per_page: true

## add "mathjax" in articles
---
title: XXX
date: 
categories:
tags:
mathjax: true
---

  post_views: true
